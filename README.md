# 🔥 Neon Brick Breaker (Shape Edition)

A modern, futuristic arcade-style Brick Breaker game built using pure HTML, CSS, and JavaScript.

Unlike traditional brick breaker games, this version uses **unique geometric shapes instead of boxes**, making the gameplay more dynamic and visually engaging.

---

## 🎮 Game Features

- 🌀 Unique shape-based targets (circles, hexagons, abstract forms)
- ⚡ Smooth ball physics and collision system
- 💥 Particle explosion effects on hit
- 🔥 Neon glowing UI and futuristic design
- 🎯 Score system with real-time updates
- ❤️ Lives system
- ⚡ Power-up system (multi-ball, slow motion, big paddle, etc.)
- 🎮 Responsive controls (keyboard, mouse, touch)

---

## ⚡ Power-ups Included

- Multi-Ball
- Big Paddle
- Slow Motion
- Explosive Ball
- Shield System
- Laser Mode (optional upgrade)

---

## 🎨 Visual Style

- Dark neon cyberpunk theme
- Glow effects and gradients
- Smooth animations and transitions
- Particle-based visual feedback

---

## 🧠 Gameplay Concept

Instead of breaking simple bricks, players interact with **dynamic floating shapes** arranged in creative patterns like:
- Spirals
- Waves
- Orbits
- Random floating clusters
